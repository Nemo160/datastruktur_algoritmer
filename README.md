# Data Structures & Algorithms Labs

Lab work from a university Data Structures and Algorithms course, implemented in C. Covers
sorting/searching, tree structures, and graph algorithms. Some skeleton/test code was provided
by the course; the core algorithmic logic is my own implementation.

## Lab 1 — Sorting, Searching & Complexity

- Implemented bubble sort, insertion sort, quicksort, linear search, and binary search
- Compared algorithm behavior across input size/order, connecting theoretical time complexity
  to measured runtime

## Lab 2 — Binary Search Trees & AVL Trees

- Implemented insertion, search, traversal, and height calculation for binary search trees
- Extended into AVL trees with rotation-based rebalancing after insertions

## Lab 3 — Graph Algorithms

- Implemented Warshall's and Floyd's algorithms (reachability / all-pairs shortest paths)
- Implemented Dijkstra's algorithm (single-source shortest path) and Prim's algorithm
  (minimum spanning tree)

## Tech stack

C

## What I learned

- How sorting/searching algorithms behave differently depending on input size and order
- How AVL trees use rotations to stay balanced, and why that matters for performance
- How greedy (Dijkstra's, Prim's) and dynamic-programming-style (Floyd's) approaches apply to
  graph problems
- Reasoning about algorithm correctness and complexity
